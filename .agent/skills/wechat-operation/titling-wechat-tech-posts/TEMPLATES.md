# Templates — Titling WeChat Tech Posts

## 同主题20标题 Prompt（10种结构）
```text
围绕主题《{主题}》，生成20个公众号标题，至少覆盖这些结构：
- 数字清单型（“5步…/7个坑…”）
- 对比冲突型（“别再用A了，用B才对”）
- 结果导向型（“让XX提升30%的方法”——如如无数据则用“明显/显著”）
- 场景共鸣型（“当你在…时，其实…”）
- 反常识型（“你以为…其实…”）
- 工具模板型（“我把模板/提示词/表格打包好了”）
并标注：适合的读者层级、点击诱因点。
```

## 克制但高点击改写 Prompt

```text
把下面标题改写成更克制但更想点的版本（给10个）。
原标题：{原标题}
要求：不夸大、不过度情绪化，但要更具体（明确对象、场景、收益）。
```
