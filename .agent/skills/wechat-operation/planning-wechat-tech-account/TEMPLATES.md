# Templates — Planning WeChat Tech Account

## 定位 Prompt（3分钟定调）
```text
请作为公众号运营顾问，帮我把账号定位写清楚：
输入：{我擅长的领域}、{我能提供的独特价值}、{目标读者}、{读者最常见3个问题}
输出：
- 一句话定位（<=20字）
- 3个内容支柱（每个支柱给5个选题）
- 风格指南（语气、专业深度、例子来源、禁忌）
- 关注理由（读者为什么必须关注我：3条）
```

## 读者画像 Prompt（精准戳痛点）

```text
请把我的目标读者拆成3类（初级/中级/决策者），每类输出：
- 他们最常用的场景
- 他们最怕的坑
- 他们最想要的“确定性”
- 他们能接受的技术细节上限
然后给出：同一主题分别怎么写（3个差异化角度）。
主题：{主题}
```

## 栏目化 Prompt（长期爆款的“系列感”）

```text
为公众号{账号定位}设计3个长期栏目：
每个栏目输出：
- 栏目名（10字内）
- 固定结构（每期都一致的骨架）
- 适合的篇幅与节奏
- 10期选题清单
要求：栏目之间差异明显（科普/实战/观点各占一个）。
```
