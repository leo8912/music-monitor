# Templates — Iterating WeChat Tech Posts

## 结尾 Prompt（不尬的关注引导）
```text
为文章《{主题}》写3种结尾版本：
A) 实用派：总结+行动清单+留言提问
B) 观点派：升维一句+反问互动
C) 福利派：提供模板/清单（但不硬广）
要求：自然、克制、像朋友提醒，不像营业员推销。
```

## 发布前质检 Prompt（主编那一套）

```text
你是公众号主编，请对这篇文章做发布前质检并给出修改建议：
- 结构是否清晰（哪里跳）
- 读者收益是否明确（哪里空）
- 术语是否过多（给替换建议）
- 例子是否贴近读者（哪里虚）
- 是否存在不确定事实（列出并要求标注或删除）
最后输出：一版“精修后的全文”（保持原意但更好读）。
文章：{粘贴全文}
```

## 一句话变锋利 Prompt

```text
把下面段落改写得更锋利、更有信息密度，但不要更难懂：
- 删除空话套话
- 多用动词
- 一句话只表达一个意思
段落：{粘贴段落}
输出：改写版 + 你删掉/替换了哪些冗余点（列表）
```

## 数据复盘 Prompt（从结果反推）

```text
这是文章发布数据：
- 阅读量{阅读量}，完读率{完读率}，分享{分享}，收藏{收藏}，留言{留言}
- 标题{标题}，主题{主题}，发布时间{发布时间}
请你分析：
1) 这篇为什么表现这样（列出最可能的3个原因）
2) 下一篇怎么改（标题/开头/结构/长度/配图/CTA各给1条）
3) 基于本篇延展3个“更有机会”的后续选题（含标题）
```
