# Templates — Formatting WeChat Tech Posts

## 配图脚本 Prompt
```text
为文章《{主题}》给出6-8张配图建议：
每张包含：位置（放在哪一段后）、图类型（流程图/对比图/清单图/示意图）、图里要表达的关键信息、图中文字（<=12字/条）。
风格：科技感、干净、少字。
```

## 公众号排版 Prompt（把草稿变可读）

```text
请把下面文章改成适合公众号阅读的排版版本：
- 小标题更短、更有信息量
- 关键句加粗（每屏不超过2处）
- 长段落拆分（每段<=4行）
- 适度加入分隔线与emoji（克制，不幼稚）
- 清单化表达优先
文章正文：{粘贴正文}
```
