# Templates — Ideating WeChat Tech Topics

## 30个选题 Prompt（含评分）
```text
你是科技公众号选题策划。围绕{领域}，生成30个适合公众号的选题。
每个选题输出：
- 标题雏形（1句）
- 读者收益（学到什么/省什么/避什么坑）
- 难度（1-5）
- 爆款潜质（1-5）及原因（是否强痛点/强对比/强争议/强清单）
- 适合的内容形式（教程/清单/测评/案例/观点）
额外：给10个“系列化栏目”名称与每个栏目3期规划。
```

## 反常识+可落地 Prompt

```text
请给我10个选题，要求同时满足：
1) 反常识或打破误区
2) 能落地（给步骤/清单/模板）
3) 与{读者画像}强相关
输出每个选题的：误区句、正确句、落地清单标题。
```
