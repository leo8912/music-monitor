# WeChat Operation Agent Skill Pack

这是一套专为科技/技术类微信公众号运营设计的 Agent Skills 集合。它覆盖了从账号定位、选题策划、标题创作、大纲构建、正文写作、案例复盘、排版优化到数据迭代的全流程。

## 目录结构

本 Skill Pack 包含以下 8 个核心 Skill：

1.  **[planning-wechat-tech-account](./planning-wechat-tech-account/SKILL.md)**: 账号定位与规划。用于定义公众号的定位、读者画像、内容支柱及长期栏目。
2.  **[ideating-wechat-tech-topics](./ideating-wechat-tech-topics/SKILL.md)**: 选题策划。生成具有爆款潜质的技术选题、内容月历及反常识角度。
3.  **[titling-wechat-tech-posts](./titling-wechat-tech-posts/SKILL.md)**: 标题创作。提供多种结构化标题方案，实现“克制但高点击”的效果。
4.  **[outlining-wechat-tech-posts](./outlining-wechat-tech-posts/SKILL.md)**: 大纲构建。构建适合公众号阅读的大纲，包含读者收益句、踩坑清单及速读版。
5.  **[writing-wechat-tech-posts](./writing-wechat-tech-posts/SKILL.md)**: 正文写作。实现一键成稿，包含钩子开头、分节写作模板及配图排版建议。
6.  **[evidencing-wechat-tech-posts](./evidencing-wechat-tech-posts/SKILL.md)**: 案例与事实核查。编写可信的案例复盘，处理数据引用并提供事实核查清单。
7.  **[formatting-wechat-tech-posts](./formatting-wechat-tech-posts/SKILL.md)**: 排版与配图优化。将“论文感”转化为“公众号感”，提供配图脚本与排版建议。
8.  **[iterating-wechat-tech-posts](./iterating-wechat-tech-posts/SKILL.md)**: 迭代与质检。提供结尾 CTA、发布前质检、句子锐化及基于数据的复盘。

## 使用方法

每个 Skill 目录下都包含两个核心文件：
- `SKILL.md`: 定义了 Skill 的触发条件、输入输出要求、工作流及质量保障。
- `TEMPLATES.md`: 提供了可直接复制使用的具体 Prompt 模板。

通过将这些 Skill 集成到 Agent 中，可以大幅提升技术文章的生产效率与内容质量。

## 核心原则

- **具体性**: 避免空泛，优先提供可执行的步骤与清单。
- **可读性**: 针对公众号阅读习惯优化，强调段落短、重点突出。
- **严谨性**: 强调事实核查，不编造数据，确保技术深度的同时易于理解。
