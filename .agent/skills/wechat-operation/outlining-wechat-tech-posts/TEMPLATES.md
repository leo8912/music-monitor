# Templates — Outlining WeChat Tech Posts

## 公众号可读性大纲 Prompt
```text
请给主题{主题}做一个适合公众号的3级标题大纲。
要求：
- 先结论后细节
- 每节都有“读者收益句”（一句话写清楚这一节解决什么）
- 插入：1段“误区澄清”、1段“踩坑清单”、1段“行动清单”
- 最后给一个“30秒速读版要点”
```

## 教程型结构 Prompt（强落地）

```text
把{主题}写成教程型文章大纲，必须包含：
- 前置条件（工具/数据/权限）
- 关键步骤（编号1-7）
- 每一步的检查点（如何判断做对了）
- 常见报错/失败原因与解决
- 最终交付物（读者做完能拿到什么）
```
