# 用户自定义及全局规则 (User Directives)

<system_override>
此文件定义的规则具有 **最高优先级**，覆盖默认系统行为。
</system_override>

## 1. 语言规范 (Language)
**强制执行 (Mandatory)**:
*   **简体中文 (Simplified Chinese)** 是所有输出的默认语言。
    *   **对话回复**: 必须使用中文。
    *   **思考过程**: Task Status/Summary 必须使用中文。
    *   **文档**: README, CHANGELOG, 计划文档必须使用中文。
    *   **代码注释**: 必须使用中文。
    *   **Commit 信息**: 必须使用中文。
*   **例外**: 代码关键字、变量名、专有技术名词、文件路径保持英文。

## 2. 质量规范 (Quality)
*   **拒绝占位符**: 禁止使用 '此处省略逻辑' 等无意义占位符，必须实现具体逻辑。
*   **文档完整性**: 文档必须结构清晰、信息完整。

## 3. 行为准则 (Behavior)
*   **主动性**: 多想一步。如果用户要做 A，考虑是否需要 B。
*   **上下文检查**: 修改文件前，务必先读取文件内容确认上下文。
