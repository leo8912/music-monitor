# Music Monitor 前端项目

本目录包含 **Music Monitor** 的前端代码，基于 **Vue 3**, **Vite**, 和 **Naive UI** 构建。

## 🛠️ 开发指南

### 项目安装

```sh
npm install
```

### 开发模式启动 (热重载)

```sh
npm run dev
```

### 生产环境构建 (编译与压缩)

```sh
npm run build
# 构建产物将生成在 dist/ 目录，生产环境中由 FastAPI 后端提供服务。
```

## 📁 目录结构

- `src/`: Vue 源代码
- `src/components/`: 可复用的 UI 组件
- `src/views/`: 页面视图 (仪表盘、设置等)
- `src/api/`: API 客户端服务
